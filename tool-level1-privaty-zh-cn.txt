【数据收集和使用】
[Data Collection and Use]

本应用是一款纯离线的补光灯工具应用，不会收集、存储或传输任何用户个人数据。
This app is a purely offline filler tool app and does not collect, store or transmit any personal user data.

【权限说明】
[Description of permissions]

本应用需要以下权限：
1. 屏幕亮度控制权限：用于调节屏幕亮度实现补光效果
2. 保持屏幕常亮权限：防止在使用补光功能时屏幕自动关闭
This app requires the following permissions:
1. Screen brightness control permission: used to adjust the screen brightness to achieve the fill light effect.
2. Keep the screen always lit permission: to prevent the screen from automatically turning off when using the fill light function.

【数据存储】
Data storage

- 应用的所有设置（如亮度值、色温值、当前模式等）仅保存在设备本地
- 卸载应用时，所有设置数据将被完全删除
- All settings of the app (such as brightness value, color temperature value, current mode, etc.) are only saved locally on the device.
- When uninstalling the app, all settings data will be completely deleted.
